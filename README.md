# Speedometer EZ
Simple GPS Speedometer with Trip Meter functions
- Counts time/distance and saves to trip log
- Configurable scale: up to 300 km/h or mph
- Only 2 buttons - start/reset trip meter and start new subpath
- Color Presets
- Speed Alerts
- Movements stats
